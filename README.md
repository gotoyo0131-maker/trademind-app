# 📈 專業交易日誌 (TradeMind Pro - Cloud)

這是一款為追求卓越績效的交易者設計的專業日誌系統。現在已全面升級至 **Supabase 雲端架構**，支援跨裝置同步、AI 心理診斷與智慧風險控管。

## 🌟 核心特色

### 1. 雲端同步與安全性
- **Supabase Auth**：使用真實 Email 註冊與授權，保障資料隱私。
- **即時同步**：資料儲存於 PostgreSQL，多裝置登入即時更新。
- **數據管理**：支援完整的 CRUD 操作，您可以新增、編輯及刪除交易紀錄。

### 2. 智慧交易工具
- **AI 心理導師**：整合 Gemini API，針對交易弱點提供犀利的心理病灶分析。
- **自動化風控計算**：輸入進場、止損與數量，系統自動計算建議風險金額與 R:R 比。
- **圖解存檔**：支援多張交易截圖雲端化儲存（URL 方式）。

### 3. 深度心理分析
- **情緒量化**：分析「焦慮」、「FOMO」等情緒對盈虧的具體影響。
- **執行紀律評分**：視覺化展示知行合一的成熟度。

## 🚀 快速開始

### 帳號登入
1. **內部授權制**：本系統目前採邀請制。請聯繫管理員建立授權後，使用指定 Email 登入。
2. **誠實記錄**：每一筆紀錄都是您進步的基石。

## 🛠 技術架構
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **AI**: Google Gemini-3-Pro-Preview
