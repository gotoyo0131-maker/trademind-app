
# 📈 專業交易日誌 (Professional Trading Journal)

這是一款為追求卓越績效的交易者設計的專業日誌系統。不僅記錄盈虧，更專注於**交易心理**、**執行紀律**與**行為數據量化**。

## 🌟 核心特色

### 1. 個人化專屬空間
- **動態命名**：系統自動根據您的用戶名生成專屬標題（如：`RURU 交易日誌`）。
- **沈浸式體驗**：專為交易者設計的深色模式登錄頁面與簡潔高效的後台界面。

### 2. 多維度數據看板
- **盈虧日曆**：直觀查看每日戰果，點擊日期即可查看當天所有明細。
- **持倉效率分析**：自動計算獲利交易與虧損交易的平均持倉時間，優化您的交易節奏。
- **多空分佈**：分析您的交易偏好，避免單一方向的思維陷阱。

### 3. 心理與行為量化分析
- **紀律指數**：基於您的執行評分自動計算「知行合一」百分比，量化您的操作品質。
- **情緒關聯圖**：分析哪些情緒標籤（如：焦慮、恐懼、貪婪）最常導致您的虧損，找出心態上的致命傷。

### 4. 數據安全性與保障 (New!)
- **一鍵備份**：支援將所有交易紀錄、策略設置與標的清單打包匯出為 `.json` 檔案。
- **數據還原**：更換設備、清除瀏覽器快取或網站重新發布後，可透過匯入備份檔快速還原完整數據。
- **完全離線**：系統不依賴外部資料庫或 AI API（如無特別開啟），確保數據處理的極致穩定與流暢。

### 5. 管理員控制台
- **用戶管理**：支持多用戶使用，管理員可凍結、刪除或修改用戶權限。
- **安全保護**：內置管理員帳號保護機制，防止誤刪或停用當前運行的管理帳戶。

## 🚀 快速開始

### 預設管理員帳號
- **用戶名**: `admin`
- **密碼**: `123`

> *註：登入後請務必在系統設置或管理後台修改您的安全密碼。*

## 🛠 技術架構
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS (現代化響應式設計)
- **Charts**: Recharts (高效能數據可視化)
- **Icons**: FontAwesome 6
- **Storage**: LocalStorage + JSON File System

## 🔒 數據隱私
本應用目前採用 **LocalStorage** 進行數據存儲。您的所有交易紀錄、圖片連結與心理備註皆保存在您的瀏覽器本地環境中，確保隱私不外洩。

**⚠️ 重要提示：由於資料保存在瀏覽器本地，建議您每週至少執行一次「系統設置 > 匯出完整資料」，並將備份檔保存於雲端碟或外部裝置。**
